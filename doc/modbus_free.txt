modbus_free(3)
==============


NAME
----
modbus_free - free a libmodbus context


SYNOPSIS
--------
*void modbus_free(modbus_t *'ctx');*


DESCRIPTION
-----------
The *modbus_free()* function shall free an allocated _modbus_t_ structure.


RETURN VALUE
------------
There is no return values.
